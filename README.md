# Bookr marketing site

- ~~Using Bulma CSS framework~~
- SASS with auto-prefixing
- Babel for ES6
- Based on `bulma-start` ([repo](https://github.com/jgthms/bulma-start))

## Install

`npm install`

## Start dev server

`npm start`

## Build for deploy

`npm deploy`

## See all commands

`npm run`
